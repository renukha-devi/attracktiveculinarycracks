# 🍳 Culinary Cracks

> **Your indecision-proof kitchen sidekick.** Decide what to cook in 60 seconds — based on your mood, ingredients, time, and gadgets.

Culinary Cracks is a smart, single-page recipe-suggestion web app built for anyone who has ever stood in their kitchen wondering *"what should I cook today?"*. Whether you're hosting a party, rushing through a workday, feeling lazy on a Sunday, or planning drinks with friends — Culinary Cracks helps you decide fast, with zero scrolling and zero clutter.

## ✨ Features

- 🎯 **5-Step Smart Questionnaire** — Tells us your occasion, food culture, available ingredients, time constraints, and cooking gadgets
- 🍲 **Curated Recipe Engine** — Built-in matching algorithm picks the best recipe from 20+ hand-picked dishes
- 📖 **Step-by-Step Manual** — Easy, beginner-friendly cooking instructions for every recipe
- 💖 **Save Your Favorites** — Bookmark recipes you love (persisted in your browser via localStorage)
- 🎵 **Ambient Background Music** — A calming Web Audio-generated lounge pad to set the cooking mood (toggle on/off anytime)
- 📱 **Single-Screen Layout** — No scrolling, no navigation menus — just answers
- ⚡ **Zero Setup** — Pure HTML, CSS, and vanilla JavaScript in a single `index.html` file
- 🚀 **GitHub Pages Ready** — Host it free, share it anywhere

---

## 🎉 How It Works

1. **Pick your occasion** — Party 🎉 · Normal 🍽️ · Urgent ⏰ · Drinks 🍹 · Lazy 😴 · Work 💼
2. **Choose your food culture** — Indian, Italian, Chinese, Mexican, Continental & more
3. **Select available ingredients** — From what's actually in your fridge
4. **Set your time limit** — From 10-minute fixes to leisurely cooks
5. **Pick your gadgets** — Oven, air fryer, stovetop, microwave, etc.

Then sit back as Culinary Cracks suggests the perfect dish with full step-by-step instructions. Love it? Save it to favorites with one click.

## 🚀 Getting Started

### Option 1: Run Locally
1. Download `index.html`
2. Double-click to open in any modern browser
3. Start cooking!

### Option 2: Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `culinary-cracks`)
2. Upload `index.html` to the root of the repo
3. Go to **Settings → Pages**
4. Under **Source**, select the `main` branch and `/root` folder
5. Click **Save**
6. Your site will be live at `https://<your-username>.github.io/culinary-cracks/` in ~1 minute
